Common Attributes

These attributes are available to all ftd kernel components.

cursor: String

You can set the cursor to be shown over any element by using cursor attribute:

this row has pointer as cursor
-- ftd.row:
width: fill
border-width: 5
border-color: yellow
padding: 10
cursor: pointer


-- ftd.text: this row has pointer as cursor
align: center
width: fill

We support the same format as CSS cursor.

link: String

This converts the element to a hyper link.

-- ftd.text: google
link: https://www.google.com

This will render like this:

google
open-in-new-tab: bool, default=False

If link is provided, this attribute can also be set to open the link in new tab instead of current tab.

-- ftd.text: google
link: https://www.google.com
open-in-new-tab: true

Which will render like this:

google (opens in new tab)
submit: Option<String>

If submit is passed, clicking on it issues a POST request on the provided URL.

-- ftd.text: test post
submit: https://httpbin.org/post?x=10

Renders as:

test post

Note: Be careful about CSRF when using this feature. If the URL is dynamically generated, include some CSRF token for example.

Note: both link and submit can not be provided.

region

This is the ARIA Region role that UI element is playing. Valid values are:

h0, h1, till h7
-- ftd.text: hello
region: h1

This tells this is a heading with the given level.

border-width

Use this property to specify the width of the border. By default the border-width is zero, and is not visible.

specifying border width
-- ftd.text: hello
border-width: 1

We can also set border of individual edges.

background-gradient

To add gradient please use the below gradient properties.

gradient-direction: Direction

Below are the supported Direction type

  • bottom to top
  • top to bottom
  • left to right
  • right to left
  • bottom-right to top-left
  • bottom-left to top-right
  • top-right to bottom-left
  • top-left to bottom-right
  • center
  • angle Integer
gradient-colors: List Color
Code sample.
Left to right gradient
-- ftd.row:
width: 400
height: 200
gradient-direction: left to right
gradient-colors: red , blue
Output

You can also make a gradient diagonally. Following example shows a gradient that starts at bottom left and goes to top right.

Diagonal Gradient
-- ftd.row:
width: 400
height: 200
gradient-direction: bottom-left to top-right
gradient-colors: yellow, organge
Output

Gradient with multiple colors

Multiple colors.
-- ftd.row:
width: 400
height: 200
gradient-direction: left to right
gradient-colors: red, green, blue, yellow, black
Output

Radial gradient that starts from the centre.

Radial gradient
-- ftd.row:
width: 400
height: 200
gradient-direction: center
gradient-colors: red, green
Output

For more control you can use angle instead of the pre-defined directions. A value of 0deg is equivalent to bottom to top. A value of 90deg is equivalent to left to right. A value of 180deg is equivalent to top to bottom.

With Angle
-- ftd.row:
width: 400
height: 200
gradient-direction: angle 90
gradient-colors: red, green
Output

More examples with angle

With 45deg Angle
-- ftd.row:
width: 400
height: 200
gradient-direction: angle 45
gradient-colors: red, green
Output
background-image: String

background-image accepts a url as the value. Use this property to make an image as background of a container.

Container with background image
-- ftd.row:
width: fill
height: 300
background-image: https://imgur.com/oCHWQQF.jpg

-- ftd.text: Sample Text
size: 30
align: center
color: white
Output
Sample Text
background-repeat: boolean

If you are using background-image property, you can also background-repeat property to true to repeat the image until the container is filled.

This property is usually helpful when you have a small image of a pattern you want to fill the container with that pattern.

background image with background-repeat
-- ftd.row:
width: fill
height: 300
background-image: https://imgur.com/LnJ4ziC.png
background-repeat: true


-- ftd.text: Sample Text
size: 30
align: center
color: white
width: fill
height: fill
Output
Sample Text
background-parallax: boolean

To achieve parallax effect on your container. Make background-parallax property to true.

Container with parallax effect
-- ftd.row:
width: fill
height: 300
background-image: https://imgur.com/oCHWQQF.jpg
background-parallax: true

-- ftd.text: Sample Text
size: 30
align: center
color: white
Output
Sample Text
